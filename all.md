2.4.1 投前策略 【数据前置线索价值】@袁兵 @魏谦丞 
Takeaway
思考：投前：甲方视角
结论：每个行业广告主，每一个新市场的进入，都需要数据来回答"值不值得投"和"怎么投"
卖点：数据丰富度，规模，实时性，准确性，完整性
结论：
广告主做出海营销，不管是电商卖货、游戏推广还是短剧分发，本质上都是同一件事——到一个新市场去获客、增长。提升品牌声量也好，拉新转化也好，最终都要回到"花钱买用户"这个动作上。
钱该往哪花？花多少？怎么花？在广告主面对一个新的国家、新的地区、新的品类时，最先要回答这三个问题。例如游戏行业来讲，游戏出海，买量就是生命线。付费获客占比远超 50%，而且这个比例还在持续上升。一款游戏能不能成，很大程度上取决于买量策略对不对。而买量策略对不对，取决于投前的市场判断准不准。

汇总所有本质解决同样的问题：
1. 信息不透明:客户看不到市场真实情况。
2. 决策太慢:从发现机会到执行太慢。
3. 试错太贵:广告、选品、达人合作、App增长都很烧钱。
4. 平台太碎片化:多平台、多地区、多角色数据无法整合。
5. 增长不可复制:成功案例无法快速复盘并复制。
他们做得好的值得我们学的
1. 市面上的第三方数据工具，核心都在卖 benchmark + 素材/竞品追踪 + 选品/达人/区域渠道拆解，头部产品已做到年收入约 50万-900万美元，说明投前数据需求已被验证。
2. 客户真正愿意付费的不是“更多数据”，而是更快、更准、更全地回答“值不值得投、该往哪投”，其中 EC 类约 30%、AG 类约 50% 已经在为这类确定性买单。
3. 我们的最佳切入点不是卖单点数据，而是卖“少花错钱的可靠结论”，用 一手投放数据 + 公域全网数据 帮客户降低试错成本、提升 ROI 下限。
4. 以投前洞察为入口，先做 订阅报告/API/专题分析，再延展到 投中诊断 + 投后归因，把一次性数据收入升级为持续性产品收入与高客单服务收入。
该怎么走我们自己的路
1. 第一步，先把数据从“展示型”升级为“判断型”：不要只告诉客户市场上有什么广告在跑，而要像 Minea、AdSpy 一样，用算法识别出真正值得关注的爆款，核心是回答“哪些广告是真的跑得动、值得抄”。
2. 第二步，按重点赛道做垂直化深挖：优先选择短剧、手游、电商等高投放行业，补充行业专属字段和模型，把通用情报做成“行业可直接决策”的情报产品。
3. 第三步，从“给客户看数据”走向“帮客户直接产出动作”：在爆款识别基础上，结合 AI 脚本生成、AI 混剪、AI 素材改写，让客户从“看懂爆款”进一步走到“快速复刻爆款”。
4. 第四步，形成产品路径闭环：先用投前情报切入，建立客户对“少花错钱”的认知，再逐步延展到创意生产、投放优化和效果复盘，把数据能力升级成完整增长解决方案。

【投前数据洞察的商业价值】
从商业角度看，投前数据洞察的价值空间非常大：
  1. 客户愿意为"确定性"买单
    1. 客户怕的不是花钱，是花错钱。投前判断准，省下的浪费就是我们可讲清的价值。
  广告主最怕的不是花钱，而是花错钱。一个错误的市场判断可能浪费几百万甚至上千万的预算。如果我们的数据洞察能帮广告主在投前就做出更准确的判断，这个价值是可以量化的——避免的浪费就是创造的价值。
  2.  数据变成服务的空间
  数据本身是原材料，但数据 + 行业理解 + 策略建议 = 高价值服务。比如：
    - 为客户出具市场进入可行性报告（数据 + 分析（结论） + 建议）
    - 提供周度/月度行业 benchmark 报告（持续订阅价值）
    - 针对具体客户的投前策略咨询（高客单价服务）
  3. 我们的优势
        我们通过私域一手媒体投放数据 + 公域达人/品类/竞品数据的双重能力，帮广告主在投放之前就看清市场全貌，制定精准的买量计划和预算分配，把"拍脑袋决策"变成"数据驱动决策"。而市面上的投前数据工具，要么只有公域数据没有一手投放数据（SocialPeta、FastMoss），要么有一手数据但没把数据洞察做成核心产品（飞书逸途、BI4Sight），要么数据来源维度不同（XMP 是广告网络侧而非代理商侧）。真正把"代理商一手媒体数据 + 公域全网数据"结合起来，做成独立的投前数据洞察产品的，目前市场上还没有。这就是我们的机会。
  4. 投前是整个营销链路的起点
   投前数据洞察不是一个孤立的产品，它是整个营销数据解决方案的入口。客户在投前用了我们的数据，投中自然会用我们的投放工具和诊断能力，投后会用我们的归因和分析。投前是获客的钩子，投中投后是留存和增收的引擎。
