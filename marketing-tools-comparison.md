## 关键指标说明

| 指标 | 计算方式 | 数据来源 |
|------|----------|----------|
| **数据规模** | 核心数据库总量 (达人/商品/素材/App) | 官方公开口径 |
| **覆盖平台** | 支持的数据源平台数量 | 产品官网 |
| **国家覆盖** | 支持分析的国家/地区数量 | 产品文档 |
| **更新频率** | 数据更新周期 (小时) | 实测 + 官方说明 |
| **数据准确率** | 核心指标相对准确度 | 第三方评测 + 用户反馈 |
| **完整性评分** | 数据发现→分析→转化闭环覆盖 | 5 分制评估 |
| **性价比** | 功能/价格比 | 10 分制主观评估 |
| **综合得分** | (准确率×0.3 + 完整性×0.3 + 规模×0.2 + 性价比×0.2) | 加权计算 |

## 数据局限性

⚠️ **注意**: 以下数据为估算值，实际使用需结合具体需求验证:

1. **数据规模**: 第三方工具多为模型估算，非官方精确数据
2. **准确率**: 基于公开评测和用户反馈，实际准确度因场景而异
3. **消耗/转化数据**: 大多数工具无法获取真实广告消耗和成交数据
4. **跨平台数据**: 单一工具难以覆盖全平台，建议组合使用
